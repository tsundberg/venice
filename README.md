# Venice

Monitoring of selected applications

## Build

    ./gradlew clean stage --daemon
    
## Run

    java -jar ./build/libs/venice-1.0-SNAPSHOT-all.jar

## Watch

The application is now running at [http://localhost:4567](http://localhost:4567)
